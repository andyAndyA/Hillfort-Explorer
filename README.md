# Hillfort Explorer

### Description
Kotlin mobile application made in Android Studio; Hillfort Explorer. This is an app that allows archaeology students in Ireland to manage and explore Hillforts to their desires.

### Technologies Used
* **Primary Language**: Kotlin
* **IDE**: Android Studio
